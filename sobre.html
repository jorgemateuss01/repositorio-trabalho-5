Sobre o Jogo

Meu Jogo Incrível foi desenvolvido com o objetivo de unir diversão e aprendizado em um só lugar.

O projeto foi criado por Jorge Mateus, com foco em desenvolver habilidades de programação, design e criatividade.

Tecnologias usadas

• HTML

• CSS

• CONSTRUCT

• JAVA SCRIPT

Objetivo do projeto

Criar uma experiência simples, divertida e acessível para qualquer pessoa jogar diretamente pelo navegador.
